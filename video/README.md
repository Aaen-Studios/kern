# kern promo video

A ~20s marketing video for kern, built with [Remotion](https://remotion.dev).
Standalone project — intentionally separate from the Tauri app so Remotion's
render dependencies (headless Chromium) stay out of the app bundle.

## What's here

- **`KernVideo`** — the main ~20s promo (1920×1080, 30fps). Four scenes:
  intro (radar glyph + wordmark) → universality (server chips register) →
  telemetry (live reactor panel) → outro (logo lockup + tagline).
- **`HeroLoop`** — a 6s seamless loop of the radar glyph idling, for landing-
  page hero backgrounds.

Both compositions reuse kern's actual brand tokens (from
`documentation/DesignGuide.md`) and port the in-app dot-matrix shaders
(`src/components/matrix/shaders/`) so the video looks like kern.

## Commands

```bash
bun install          # install deps
bun run dev          # open Remotion Studio (live preview + timeline)
bun run sfx          # (re)generate procedural SFX WAVs → public/audio/
bun run render       # render both MP4s → out/
bun run render:promo # just the main promo
bun run render:loop  # just the hero loop
```

## Audio

All audio is procedurally synthesized — fully self-contained, no licensing.

- **Background music** (`public/audio/music.wav`) — a dark ambient-tech bed at
  120 BPM: sub-bass pulse on the beat, a minor pad progression (Am/F/C/G),
  arpeggiated "radar ping" blips, and a noise riser under the outro. Generated
  by `scripts/gen-music.ts`. Re-run `bun run music` to regenerate. To swap in a
  real track, replace `music.wav` with a ~120 BPM file of the same name;
  animations are beat-synced (`FRAMES_PER_BEAT = 15`) so a matched track slots
  in cleanly.
- **Procedural SFX** (`public/audio/sfx-*.wav`) — blips, a whoosh, a resolve
  chord. Generated by `scripts/gen-sfx.ts` via `bun run sfx`.
- **Everything**: `bun run audio` regenerates music + SFX together.

## Structure

```
src/
  theme.ts            # brand tokens (single source of truth)
  fonts.ts            # JetBrains Mono loader
  audio.ts            # SFX paths + scene/beat timing
  lib/shaders.ts      # ported polarRadar, sineRipple, reactorChannel
  components/
    DotGrid.tsx       # dot-matrix canvas renderer
    RadarGlyph.tsx    # the logo mark
    Caption.tsx       # kinetic typography
    Wordmark.tsx      # "kern" + tagline lockup
    ServerChip.tsx    # instance registration pill
    ReactorBar.tsx    # CPU/RAM activity strip
  scenes/
    Intro.tsx · Universality.tsx · Telemetry.tsx · Outro.tsx
  KernVideo.tsx       # main composition (4-scene Series + audio)
  HeroLoop.tsx        # 6s loop composition
  Root.tsx            # composition registry
  index.ts            # registerRoot entry point
scripts/
  gen-sfx.ts          # procedural SFX synthesizer
  render.ts           # programmatic render (both MP4s)
```
